2、tanh函数

tanh函数又称位双曲正切函数，其计算公式为：

函数图像为：

import numpy as np
import matplotlib.pyplot as plt
 
x =  np.arange(-10,10,0.01)
y=(np.exp(x)-np.exp(-x))/(np.exp(x)+np.exp(-x))


#tanh函数图像
plt.plot(x,y)
plt.grid(color='b' , linewidth='0.1' ,linestyle='--')


#tanh导数图像
