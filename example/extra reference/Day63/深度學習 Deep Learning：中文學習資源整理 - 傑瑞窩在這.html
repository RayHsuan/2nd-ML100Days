深度學習 Deep Learning：中文學習資源整理

2016-09-29 · Jerry · Deep Learning

大學那時，看著室友們去修「機器學習」這門課，被一堆數學算式打趴，就對這個領域有種畏懼。所以每次討論到相關的東西，我就馬上閉起耳朵，覺得好難啊，聽了應該也不會。直到現在，心境變得不一樣了！因為學分數已經足夠，我可以自由地去旁聽課程，也因為沒有壓力的學習，反倒更認真 xd

這裡整理跟深度學習有關的資源，希望有興趣的朋友們，也能一起努力！

1. 李宏毅老師：一天搞懂深度學習

之前李宏毅老師在台灣資料科學年會曾上過一堂快速入門課，題目是「一天搞懂深度學習」，後來網路上有人濃縮成 36 分鐘的影片，講的也不錯

講義連結 (SlideShare)：http://www.slideshare.net/tw_dsconf/ss-62245351

網友整理的影片 (用 1.5 倍速看看剛好)

請支持《傑瑞窩在這》原創文章。原文標題：深度學習 Deep Learning：中文學習資源整理，原文網址：https://jerrynest.io/deep-learning-resource/

2. 李宏毅老師：深度學習課程

老師的教學十分有趣，使用精靈寶可夢、御宅動漫等題材引起學生的共鳴，讓你看課程影片就像是在看影集，介紹原理之外，還會告訴你如何應用，收穫滿滿。老師開的課程包含機器學習與深度學習，影片與講義都可以在課程網站上看到。剛開始接觸深度學習的朋友，建議先觀看 Machine Learning and having it deep and structured (2015, Fall)，接著再看 2017 開課程，裡面有較新的深度學習模型介紹，比如 Generative Adversarial Network (GAN)。

課程網站：http://speech.ee.ntu.edu.tw/~tlkagk/courses.html

3. 陳縕儂老師：深度學習應用課程

人氣破表的正妹老師開課囉，開學第一堂課同學們就擠爆上課教室！這麼多人並不是找老師加簽，而是要與老師自拍合照啊啊！老師的專長偏向語意理解、自然語言處理，所以這門課主要是用深度學習的方式做相關應用。聽了兩堂課，覺得老師講得很清楚，不時也會講幾句搞笑的話，我看大家都很有精神的專心聽課，哈哈哈

課程網站：https://www.csie.ntu.edu.tw/~yvchen/teaching

4. YouTube 頻道

網路上有很多不錯的教學影片，我覺得周莫煩 Python 以及 開發者講堂 這兩個頻道講得很清楚，可以快速吸收。其中周莫煩 Python 除了介紹深度學習外，還有 Google 開源工具 TensorFlow 的教學影片，搭配服用，更容易理解唷。

周莫煩 Python 介紹 CNN (卷積神經網路)
